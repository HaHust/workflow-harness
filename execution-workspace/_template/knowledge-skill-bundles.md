# Knowledge Skill Bundle Profiles

Use `templates/knowledge-skill-bundles.md` to construct:

- Full bootstrap A01 bundle: `repository-scan` first, selected discovery skills, `knowledge-index-update` last.
- Incremental A01 bundle: `incremental-git-scan` first, affected skills only, `knowledge-index-update` last.
- R01 review bundle: `knowledge-review`.
- A02 precedent search: `similar-code-search` only when relevant.

All bundle rows must contain concrete skill files resolved from Workflow Home and produce `Skill Files Read` evidence.
